# 玉兔6 办公室视觉重设计 · 完整方案 v1

> 经两轮确认定稿。日期 2026-06-20。样板见 `office-demo-assets/vector/office_sample_final.svg`。
> 目标:精致、不割裂、高端简约、通透,科技大厂感。**矢量 SVG 程序化生成**(可控、无糊、无拼缝、可无限清晰)。

## 一、风格定调(已确认)
- 空间:**明亮通透白调**(浅地毯 + 玻璃落地窗 + 玻璃隔断)
- 家具:**深灰 + 原木**高级感(空间浅、家具深,高级反差)
- 画风:**扁平矢量等距 SVG**
- 工位:**双屏、员工背对镜头、屏幕朝我们**(可见摸鱼);屏幕内容**按角色不同**
- 道具:**适度点缀**(键鼠 + 咖啡 + 绿植 + 台灯)
- 分区:**玻璃隔断 + 外圈落地窗**

## 二、等距规格(比例)
- 投影 2:1,地块 footprint 120×60px
- **薄地块**:厚度 4px(几乎贴地)
- **高墙**:落地窗 200px、半高玻璃隔断 118px
- 墙高 : 地块宽 ≈ **1.6:1**(科技大厂挑高通透感)
- 细描边 1.2–1.4px(`#2a323d`),扁平干净

## 四、组件清单(全套要生成)
**A. 地块**:薄短绒地毯地块(无缝平铺)
**B. 墙体**:① 玻璃落地窗(左朝向 / 右朝向)② 半高玻璃隔断 ③ 玻璃门(开 / 关)④ 墙角衔接件
**C. 工位**(双屏背对,屏幕按角色显示不同内容):
- 程序员=代码/IDE、洞察员=图表看板、监管/质量=数据流、CEO/主管=多窗管理面板、秘书=文档/日程、维修员=日志终端、Peekaboo=截图预览、Hermes=消息流、董事长=大屏总览…
**D. 家具**:办公桌(深灰+木)、办公椅(深灰、背对)
**E. 道具**:键鼠、咖啡杯、台灯、绿植

## 五、分区组装(玻璃隔断分房间)
- **总裁办公室**(董事长+秘书+CEO):独立玻璃房,整面落地窗,最高端
- **系统办公室**(主管+程序员+外包)
- **Simulaid 片区**(主管+程序员+外包)
- **公共协作区**(外围:洞察/维修/Peekaboo/Hermes/优化/自优化开发)
- 各区用玻璃隔断分隔、外圈落地窗采光 → 通透不割裂

## 六、员工动画接口(给后续摸鱼动画留好)
- 每工位含两层:**员工背影层**(坐在椅子上,背对)+ **屏幕显示层**(可替换内容)
- 摸鱼动画后续叠加:屏幕换画面(刷视频/打游戏/看股票)+ 员工背影微动(伸懒腰/喝咖啡)
- 工作态 vs 摸鱼态:屏幕内容切换 + 姿态切换
