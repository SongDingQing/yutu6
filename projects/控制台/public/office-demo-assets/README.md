# Generic office assets

This directory contains only the small asset subset used by the generic Yutu6 workspace and office preview.

- Assets are project-neutral and may be replaced by a project department pack.
- Runtime pages must provide an avatar fallback when a custom asset is missing.
- Unapproved local reference sheets, generation jobs, and project-specific art are not part of the public distribution.
- New production art must pass the template gate in `projects/控制台/templates/office-image/`.
