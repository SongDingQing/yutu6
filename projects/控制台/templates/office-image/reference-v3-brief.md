# 玉兔6办公室 V3 参考图输入稿

状态: spec-only pending。老板确认前不调用 imagegen 或 Meowa。

## 目标

生成一张统一风格参考图，用于后续所有办公室人物、地块、工位、董事长办公室和动画素材。V3 的重点不是换画风，而是修正 V2 的比例、拼接和动画输入问题。

## 总体画风

- 2:1 等距办公室视角，深色控制台调色。
- 精致、高密度、像素感明确，但不要粗糙大颗粒。
- 人物是精致 Q 版人类办公室角色，不是真实照片，也不是动物 CEO。
- 头身比比 V2 更克制: 头部约占坐姿人物总高的 28-32%。
- 所有角色、桌椅、地块、墙面共用同一光源、线条粗细、阴影方向。

## 必须包含的参考块

1. 1x1 纯色地毯地块
   - 有厚度，不是平面菱形。
   - 四个菱形边界完整，无缺角、无遮挡截断。
   - 相邻地块能无缝拼接。

2. 普通员工 2x2 工位
   - 桌子、椅子、电脑、人物作为同一比例体系设计。
   - 不是 1x1 桌子放大，也不是桌椅分开漂浮。
   - 四周 footprint 完整，能和地毯 / 其他工位拼接。
   - 程序员屏幕必须朝向员工和观察者，不能反向。

3. 董事长办公室 5x5
   - 必须是整数 5x5 格子办公室组件。
   - 主席桌 / 高背老板椅 / 董事长站位或坐位占清晰 2x3 主区域。
   - 可包含落地窗、城市天际线、多屏、沙发、书架、奖杯，但都要落在 5x5 内的整数格布局。
   - 可以作为一张整体组件图，但必须看得出 5x5 footprint 和完整边界，不能像一次性透视插画。

4. 人物参考
   - 董事长: 年轻、清秀、霸道青年总裁，黑色正装，自信但不过分写实。
   - 秘书: 职业装，坐在工位打字，双手放在键盘上。
   - 主管 / CEO / 程序员各一个，用服饰和道具区分，不改变画风比例。

5. 动画输入提示
   - 后续 Meowa 动画源图最大按 256x256 准备，避免超出 pixel engine 限制。
   - 透明背景需要干净 alpha，不允许头发、肩膀、手、椅子出现白色描边或抠图残边。
   - 打字动作不是手臂上下摆动；手掌和手腕基本固定，只做手指轻微交替敲击。

## 负面要求

- 不要 1x1 桌子伪装 2x2。
- 不要缺角、断边、被裁掉的 footprint。
- 不要把董事长办公室画成没有格子逻辑的一张小插画。
- 不要把人物放大到超过源图细节密度。
- 不要白色 halo / chroma-key fringe。
- 不要手臂大幅抖动式打字。

## 交付前检查

- 可见 1x1、2x2、5x5 三种格子尺度。
- 所有桌椅和人物比例一致。
- 5x5 董事长办公室边界完整，主区域 2x3 明确。
- 2x2 工位边界完整，可拼接。
- 打字动画说明明确为 finger-only。
