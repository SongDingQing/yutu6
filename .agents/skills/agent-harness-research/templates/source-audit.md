# Claim-to-Evidence Source Audit

| Claim ID | Local problem/opportunity | Claim | Primary evidence | Contradiction/limit | Confidence | Proposed action | Metric | Risk | Rollback | Decision |
|---|---|---|---|---|---|---|---|---|---|---|
| CLM-01 |  |  | `SRC-.. / file:line` |  | `high | medium | low` |  |  |  |  | `recommend | experiment | defer | reject` |

Reject or defer a claim when:

- it only repeats a README or secondary article;
- no local gap exists;
- expected gain cannot be measured;
- the proposal duplicates an existing gate/hook/tool;
- permission, shared-state, license, or rollback boundaries are unclear.
