# yutu6-unified-startup

玉兔6本机运行栈的统一入口。先读 `operations.md`，再按需查看
`projects/控制台/config/startup-components.json`；不要通过进程名猜测组件状态。

- 入口：`/Users/yutu6/玉兔6工作区/start-all.sh`
- 清单：`projects/控制台/config/startup-components.json`
- LaunchAgent：`com.yutu6.startup`
- 状态：`projects/控制台/artifacts/startup/status.json`
- 设计：`docs/设计/玉兔6-统一启动与重启恢复.md`
