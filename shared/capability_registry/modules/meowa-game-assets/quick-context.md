# Quick Context

- Meowa is an optional shared capability under `shared/tools/meowa/`.
- Set `YUTU6_ROOT` to the repository root before using examples in this module.
- Run `python3 "$YUTU6_ROOT/shared/tools/meowa/meowart_api.py" skill-doc --task "<brief>"` before choosing a command.
- The key name is `MEOWART_API_KEY`; its value remains in a local private env file.
- Generated assets must go to the requesting project's own artifact directory, such as `projects/<projectId>/artifacts/meowa/<taskId>/`.
