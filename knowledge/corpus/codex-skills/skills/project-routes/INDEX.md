# 项目 Skill 路由

通用发行版不预装任何具体项目路由。项目能力包安装后，应在本目录登记 `<projectId>.md`，并声明：

- 项目标识和根路径；
- 可用 wrapper/skill；
- 权威文档读取顺序；
- 独占资源与锁名；
- 测试、构建、发布和回滚命令；
- 账号、登录和外部发送的人工确认边界。

路由选择顺序：显式项目 ID → 当前目录 → 涉及文件。不能唯一确定项目时必须询问，不得借用其他项目的命令。

并发写入使用 `route_lock.py`，锁根目录默认 `/tmp/codex-skill-locks`。锁只保护项目独占资源，不能代替 Git diff、任务队列和 done gate。
