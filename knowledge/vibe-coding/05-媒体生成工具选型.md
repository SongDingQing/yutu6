# 05 媒体生成工具选型(截至 2026-07,季度刷新)
适用:secretary / frontend-designer / 涉及 meowa·gemini 生成的任务

> 注意:本章矩阵是 MKT 平台的选型,玉兔6 主用 meowa(喵爪)+其挂载的 gemini。**通用原则直接吸收,具体工具按需参考。**

## 通用原则(直接吸收)
1. **比价必先统一单位**:计费单位五花八门(/图 /百万像素 /token /秒 /credit),不换算就对比会误判几倍到几十倍。统一锚点:「元/张1K图」「元/5秒720p」,写进账本口径并标注价格日期
2. **视频永远保持双供应商**:闭源生成模型有整体下线风险(Sora 已死:2026-04 关停 App、2026-09 停 API 的实证)
3. **质量档位差价巨大**:生图档位差可达 35 倍——按用途给默认档,别全走最高档
4. 海外模型需出口链路,先在生产环境验证连通再写代码;聚合平台只用于评测对标不做生产主路径
5. 订阅制/credit 制计费模型不匹配按量平台
6. Midjourney 无公开 API,市面"MJ API"全是违 ToS 的 wrapper,账号随时被封——只做灵感来源
