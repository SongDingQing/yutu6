# Multi-Agent Collaboration Contract

This module is the shared capability and handoff contract for Hermes/Yutu, Codex, and future local agents on this Mac mini.

Use it to answer:

- Which agent should handle a task?
- What can each agent do?
- What information must be passed across agent boundaries?
- Which actions require user confirmation?
- Where are the live files and durable records?

The goal is persistent multi-agent coordination, not one-off chat memory.
