# Capabilities And Flows

## 语音唤醒

输入：

- 麦克风音频。
- 唤醒词：`玉兔玉兔`、常见误识别变体、`Hermes听令`。

输出：

- 只唤醒时说 `我在`。
- 有完整命令时进入后续流程。

## 语音回复

流程：

```text
用户语音 -> 本地 STT -> 纠错 -> 快速回复或 Hermes API -> TTS 播放 -> 飞书记录
```

飞书记录标记：

`【语音回复记录】`

记录内容：

- 时间。
- 用户语音文本。
- 玉兔回复文本。
- 纠错提示。

## 语音任务确认

触发例子：

- “给姐姐发邮件”
- “帮我发邮件”
- “这个任务交给 Codex”
- “给某个项目加功能”

输出：

- 飞书卡片：`Command Required: Voice Task`
- 按钮：`确认处理` / `取消`

确认后：

- Hermes 把这条语音任务当作已确认请求继续处理。
- 如果缺正文、主题、路径等信息，Hermes 应追问缺少项。

取消后：

- 删除 pending task。
- 卡片显示已取消。

## Codex Handoff

触发：

- 本地项目修改。
- Unity 游戏功能。
- 测试、构建、打包 APK。
- 代码审查或修 bug。

输出：

- 飞书卡片：`Command Required: Codex Handoff`

确认后：

- 启动本地 Codex CLI。
- Codex 完成后把总结发回飞书。
- 如果生成 APK 且请求要求返回 APK，会尝试发 APK 附件。

## ASR 纠错

```text
纠错：错词=>正确词
```

- 写入 `/Users/yutu/.hermes/voice-wake/asr-context.json`。
- 正确词也进入热词库。

## ASR 热词

主人在飞书回复：

```text
热词：词1、词2
```

效果：

- 写入热词库。
- 后续语音识别偏向这些词。

## 联网搜索

触发：

- 最新消息。
- 新闻。
- 联网查一下。
- 搜索一下。
- 会变动的信息。

工具：

`brave_search`

注意：

- 除非 Brave Search 工具失败，不应说没有联网能力。
