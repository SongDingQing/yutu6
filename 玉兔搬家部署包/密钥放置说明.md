# 密钥放置说明(玉兔搬家部署包 · 私人)

⚠️ 这个包是**私人**的,含或将含你的密钥 / 凭证。**绝不要把这个包发给别人** —— 给朋友的是「通用玉兔6部署」,那个不含密钥。

## 密钥放哪
- **代理节点**(v2rayN + reality 私钥 / TUIC 密码):放进 `代理/`(见 `代理/说明.txt`)。
- **API Key**(走 API 模式时):复制 `secrets.env.example` 为 `secrets.env` 填入;`secrets.env` 不进 git、不外传。
- **Claude / Codex 登录态**:想免重登,就把旧机的 `~/.claude`、`~/.codex` 拷过来。

## 更安全的做法(推荐)
- 密钥**尽量不进这个 zip** —— 装机时再从 macOS 钥匙串 / 密码管理器注入,包里只留 `.example`;
- 如果确实把密钥打进了包:当**机密文件**保管,传输用加密或当面拷,**用完即删**,别留在 U 盘或云盘。

## 这个包 vs 通用包
- **玉兔搬家部署包(本包)** = 通用框架 + 你的完整设计 + 密钥 + 精华索引 → 只给你自己。
- **通用玉兔6部署** = 脱敏框架,零密钥、零私人内容 → 给朋友。
