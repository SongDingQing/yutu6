# 玉兔6 · 长期记忆

`memory/` 保存当前安装实例提炼后的长期知识，不保存密钥、登录信息或原始对话流水。

| 文件 | 用途 |
|---|---|
| `preferences.md` | 使用者确认过的协作偏好与边界 |
| `decisions.md` | 已拍板决策及原因 |
| `experience.md` | 可复用的成功模式与失败教训 |
| `entities.md` | 已安装项目、角色、能力与机器的稳定事实 |
| `sessions/` | 经筛选的会话摘要 |

首次部署保持空白。秘书和记忆官只能增量提炼当前实例产生的内容；项目卸载后应同步清理或标注对应记忆。
